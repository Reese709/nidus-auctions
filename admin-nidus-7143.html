Nidus Auctions – Admin

This site is running on GitHub Pages, so changes are done by editing files in GitHub (static admin). This page is just for you — don’t link it publicly.

1. Change Background Image

Open style.css → find .hero → change the background-image URL to your own image (you can upload it to the repo and point to it).

2. Change Social Links

Open index.html and edit the footer links: Facebook/Instagram/LinkedIn/YouTube.

3. Add New Auctions

Open scripts.js and add a new object to the auctionItems array. 

{
  title: "5HP Baldor Motor",
  image: "https://your-image-or-github-path.jpg",
  price: "$480",
  category: "Motors"
}

Save / Commit on GitHub → refresh https://YOURUSERNAME.github.io/nidus-auctions/auctions.html

4. Make it more hidden

You can rename this file to panel-9283.html or something random so no one guesses it.
